# Hotel-Management-system
Hotel Management System is a C++ based console application designed to manage hotel room bookings efficiently. The system allows users to book rooms, check out guests, view room availability, and search guest records.

The project uses basic data structures and object-oriented programming concepts to manage room information and guest details.

Technologies Used
C++
Structures
Arrays
Console Application
Features
Room Booking
Guest Check-In & Check-Out
Room Status Display
Guest Search Functionality
Purpose

The purpose of this project is to demonstrate the implementation of a simple hotel management system using C++ programming concepts.
